Super Simple Guide: One Click DC Motor Launcher
==============================================

What you get
------------
Double-click an icon on the Pi desktop. It opens the motor server in a terminal and launches the web UI in Chromium.

Before you start
----------------
- Your project folder lives at /home/pi/Cansat-NOVAsync. If yours is somewhere else, swap that path in the steps below.
- You've already installed the Python packages (`adafruit-circuitpython-motorkit` and `flask`).
- Chromium is installed (it is by default on Raspberry Pi OS).

Step 1 – Make a tiny script that does the work
---------------------------------------------
1. Open Terminal on the Pi.
2. Type `nano ~/start_dc_control.sh` and press Enter.
3. Paste these four lines exactly:
   #!/usr/bin/env bash
   cd /home/pi/Cansat-NOVAsync || exit 1
   lxterminal -t "DC Motor Server" -e bash -c "python3 src/receiver/motor/web_dc_control_server.py --host 0.0.0.0 --http-port 8765; read"
   sleep 2 && chromium-browser http://localhost:8765 &
4. Press Ctrl+O, Enter to save, then Ctrl+X to quit Nano.
5. Type `chmod +x ~/start_dc_control.sh` so the script can run.

Step 2 – Drop a shortcut on the desktop
---------------------------------------
1. Still in the terminal, run:
   cat <<'EOF' > ~/Desktop/DC-Control.desktop
   [Desktop Entry]
   Type=Application
   Name=DC Motor Control
   Exec=/home/pi/start_dc_control.sh
   Icon=utilities-terminal
   Terminal=false
   EOF
2. Make the shortcut clickable: `chmod +x ~/Desktop/DC-Control.desktop`.

Step 3 – Use it
---------------
- Double-click the "DC Motor Control" icon on the desktop.
- A terminal window pops up and starts the web server.
- Two seconds later Chromium opens http://localhost:8765 with the buttons.
- When you’re done, close Chromium and press Ctrl+C in the terminal.

Help! Something broke
---------------------
- Double-click does nothing: run `ls -l ~/start_dc_control.sh ~/Desktop/DC-Control.desktop`. If you don’t see `-rwxr-xr-x`, re-run the `chmod` commands.
- Terminal says modules are missing: run `sudo pip3 install adafruit-circuitpython-motorkit flask` once.
- Chromium doesn’t open: maybe you removed it. Change the last line to use another browser, e.g. `firefox`.
- Different project folder? Edit the script (`nano ~/start_dc_control.sh`) and change the `cd` line to your path.

That’s it. One icon, motors ready.
